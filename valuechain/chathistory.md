# 📜 AI 노드 탐색기 개발 대화 기록 (Chat History)

본 문서는 "옵시디언 스타일 AI 노드 탐색기" 개발 과정에서 진행된 주요 요청과 작업 내용을 기록한 것입니다.

## 1. 초기 요청 및 기본 구현 (Phase 1)
- **요청**: 옵시디언 UX를 기반으로 키워드 클릭 시 관련 내용을 자동 생성하는 그래프 탐색기 제작.
- **작업**: 
    - `react-force-graph-2d`를 이용한 기초 그래프 구현.
    - Python(`TopicAnalyzer`)을 이용한 대화 데이터 분석 및 JSON 추출.
    - 사이드바 및 타이핑 효과(Typewriter effect) 구현.

## 2. 시각적 정교화 및 버전 관리 (Phase 2)
- **요청**: 노드 크기(dot size)를 옵시디언과 동일하게 축소하고 GitHub에 커밋.
- **작업**: 
    - 노드 크기 및 그래프 물리 계수(`charge`, `linkDistance`) 조정.
    - Git 저장소 초기화 및 첫 커밋 수행.
    - 원격 저장소(`KIIMM22/aigraphview`) 연동 설정.

## 3. 기능 고도화 및 트러블슈팅 (Phase 3)
- **요청**: AI를 이용한 동적 노드 확장, Imagen 컨셉 이미지, TTS 음성 안내 추가.
- **작업**: 
    - **동적 확장**: 노드 클릭 시 연관 노드 3개를 즉시 생성하는 로직 구현.
    - **멀티모달**: Imagen 스타일의 추상화 이미지 표시 및 오디오 가이드 UI 구현.
    - **트러블슈팅**: Tailwind CSS v4 호환성 이슈 해결을 위해 v3로 다운그레이드 및 PostCSS 설정 최적화.

## 4. 최종 마무리 (Phase 4)
- **요청**: 작업 내용 요약(`workaround.md`) 및 대화 기록(`chathistory.md`) 저장.
- **작업**: 
    - 전체 기술적 결정 사항과 해결책 문서화.
    - 현재 대화의 흐름을 정리하여 기록물로 남김.
